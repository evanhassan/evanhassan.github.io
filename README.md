# Personal Portfolio

Minimal static portfolio for GitHub Pages.

## Edit

- Change project text in `index.html`.
- Change colors in `styles.css` under `:root`.
- Replace placeholder `#` links with real GitHub repos or case studies.

## Host on GitHub Pages

1. Create a new GitHub repo named `evanhassan487.github.io`.
2. Upload `index.html`, `styles.css`, and this README.
3. Go to Settings → Pages.
4. Under "Build and deployment," choose "Deploy from a branch."
5. Select `main` and `/root`, then save.
6. Your site should be live at `https://evanhassan487.github.io/`.
